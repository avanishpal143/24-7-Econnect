## 🔧 Maintenance:

### To Update Menu Items:
Edit the navbar in each HTML file or update the template in `/includes/navbar-complete.html`

### To Change Colors:
Edit CSS variables in `/assets/css/navbar-glassmorphism.css`:
```css
:root {
  --nav-glass-bg: rgba(255, 255, 255, 0.15);
  --nav-text: #2d3748;
  --nav-text-hover: #800020;
  --nav-cta-bg: #1a202c;
}
```

### To Adjust Width:
Change `max-width` in `.premium-nav__container`:
```css
.premium-nav__container {
  max-width: 1280px; /* Change this value */
}
```
