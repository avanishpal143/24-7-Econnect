A modern, responsive website for India's leading bulk SMS and communication services provider.

## 🚀 Features

- **Premium Glassmorphism Navbar** - Ultra-transparent navigation with smooth animations
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Multi-Service Platform** - Bulk SMS, WhatsApp API, Google RCS, Voice IVR, Email Marketing
- **Modern UI/UX** - Clean design with smooth transitions and animations
- **SEO Optimized** - Proper meta tags and semantic HTML structure

## 🚀 Getting Started

### Local Development

1. Clone the repository
```bash
git clone <repository-url>
cd 24-7-econnect
```

2. Open with a local server
```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (http-server)
npx http-server public -p 8000

# Using PHP
php -S localhost:8000 -t public
```

3. Open browser and navigate to:
```
http://localhost:8000
```

### Deployment

Simply upload the `public` folder contents to your web server.
